Measures Taken

  • All images include descriptive alt text
  • Semantic HTML used throughout (header, nav, main, footer)
  • Keyboard navigation supported on all pages

  • Color contrast meets WCAG 2.2 AA requirements
  • Fonts are scalable and readable

Testing

  • Manual keyboard navigation testing
  • Color contrast checked via Coolors Contrast Checker
